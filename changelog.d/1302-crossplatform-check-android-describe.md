- **`cross-platform-check.sh` can cross-check the demo APK manifest ([#1302](https://github.com/sceneview/sceneview/issues/1302)).** A new opt-in `--with-apk` flag builds (or reuses) the `android-demo` debug APK and runs Google's `android describe` on it to verify the manifest's exposed activities and intent filters match expectations — exactly one `MainActivity`, a `LAUNCHER` entry, the `sceneview://demo` deep-link `VIEW`/`BROWSABLE` intent, and the `io.github.sceneview.demo` package id — then cross-checks the Android `DemoRegistry` demo count against the iOS `SamplesTab` inventory so a platform missing a demo surfaces as drift. The fast source-only path stays the default. A reusable `android_cli_describe` helper was added to `.claude/scripts/lib/android-cli.sh`. `Closes #1302`.
