- **Web demo: self-host every asset so the deployed viewer stops bricking ([#1573](https://github.com/sceneview/sceneview/issues/1573), [#1586](https://github.com/sceneview/sceneview/issues/1586)).** `samples/web-demo` loaded its 15-model GLB catalog *and* the `filament.js` / `filament.wasm` / `sceneview.js` engine from `cdn.jsdelivr.net/gh/sceneview/sceneview@main/...`. jsDelivr's gh-proxy returns HTTP 403/404 for those blobs, so `initSceneView()` never resolved and the deployed viewer at `sceneview.github.io/web-demo/` was stuck on its loading overlay (and the Playwright suite went red). All assets are now bundled under `src/jsMain/resources/models/` and `src/jsMain/resources/js/` and referenced by relative path, so they ride into `jsBrowserDistribution`, the deployed site, and the Playwright dev server with zero external CDN. Engine init is also decoupled from the first model load: a flaky model load now surfaces as a transient chip instead of a fatal "Failed to initialize" overlay. The Playwright `isIgnorableNoise` filter was tightened so a genuine local model/engine 403/404 fails the suite instead of being silenced as "CDN noise" — only Sketchfab third-party API errors remain ignorable.

- **Web demo catalog expanded with Lighting, Animation, Text, and Environment tabs ([#1362](https://github.com/sceneview/sceneview/issues/1362)).** The `samples/web-demo` playground previously exposed only Models / Geometry / Physics / Settings — a small fraction of the SDK versus Android's ~39 demos. It now ships four new tabs wired to real `sceneview.js` API calls: **Lighting** adds and removes directional/point/spot lights (`addLight` / `removeNode`), **Animation** loads self-hosted animated glTF models and drives keyframe playback (`playAnimation` / `stopAnimation`), **Text** renders billboarded 3D text nodes (`createText`), and **Environment** controls image-based lighting via spherical-harmonic presets (`setEnvironmentSH`), background color, and bloom strength. First slice of the cross-platform demo-parity effort; web-demo only.
