# architecture-3d plugin for Claude Code

MCP for architects and interior designers. Gives Claude tools to generate 3D concepts, floor plans, material palettes, lighting studies, and AR site previews.

## What you get

- **Concept generation** — text-to-3D building / room concepts.
- **Floor plans** — auto-generated layouts with rooms, doors, windows, dimensions.
- **Material palettes** — coordinated material/finish suggestions per design style (modern, brutalist, scandinavian, japandi, …).
- **Lighting studies** — sun-path simulation, IES luminaire recommendations.
- **AR site previews** — anchor a building model on a real plot via ARCore / ARKit.

## Install

```
/plugin marketplace add sceneview/sceneview
/plugin install architecture-3d@sceneview
```

## Pairs well with

- `sceneview` plugin — for shipping the generated buildings as native Android/iOS AR apps.
