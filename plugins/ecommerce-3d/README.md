# ecommerce-3d plugin for Claude Code

MCP for e-commerce 3D product visualization. Gives Claude tools to ship interactive product viewers, AR try-on, and configurators on Shopify, WooCommerce, or any custom storefront.

## What you get

- **3D product viewers** — generate GLB-ready scenes with rotation, zoom, hotspots.
- **AR try-on** — `<model-viewer>`-compatible setup for iOS Quick Look and ARCore.
- **Product configurators** — variant swapping (color, material, size) with live preview.
- **Shopify integration** — submission-ready 3D models for the Shopify 3D Viewer.
- **Optimization** — auto-LOD, texture compression, draco/meshopt ready.

## Install

```
/plugin marketplace add sceneview/sceneview
/plugin install ecommerce-3d@sceneview
```

## Pairs well with

- `sceneview` plugin — embed the generated 3D viewers natively in Android/iOS apps via Jetpack Compose / SwiftUI.
