# realestate-3d plugin for Claude Code

MCP for real-estate agents and property platforms. Gives Claude tools to generate listings, floor plans, virtual tours, and 3D property previews — bilingual EN/FR.

## What you get

- **Property descriptions** — AI-generated, MLS-ready listings in English or French.
- **Pricing analysis** — comparable-listing lookups and price-per-m² breakdowns.
- **Floor plan generation** — text-to-floorplan with room layouts.
- **Virtual tours** — 3D scene generation for the SceneView SDK or web embed.
- **AR previews** — staging models for ARCore / ARKit integration.

## Install

```
/plugin marketplace add sceneview/sceneview
/plugin install realestate-3d@sceneview
```

## Pairs well with

- `sceneview` plugin — for embedding the generated 3D tours in an Android/iOS/Web app.
