# french-admin plugin for Claude Code

MCP for French administrative paperwork. Gives Claude tools to navigate French bureaucracy — tax filings, social-security, unemployment, redundancy, official forms.

## What you get

- **Taxes** — *impôts sur le revenu*, deadlines, deductible categories, simulator.
- **Social security** — URSSAF rates, CAF benefits eligibility (APL, RSA, prime d'activité), CPAM.
- **Unemployment** — ARE rights, ARCE, training rights (CPF, France Travail).
- **Redundancy** — *licenciement* procedure, severance pay, *rupture conventionnelle*.
- **Self-employment** — *micro-entreprise* / *auto-entrepreneur* setup, thresholds, social charges.
- **Service-Public.fr** — direct lookup of official guides and procedures.

## Install

```
/plugin marketplace add sceneview/sceneview
/plugin install french-admin@sceneview
```

## Languages

Tools answer in French by default. Pass `lang: "en"` for English summaries.
