## Submission checklist

For each PR:
1. Fork the repo
2. Add the entry in the correct section (alphabetical)
3. PR title: "Add SceneView — Compose-native 3D & AR SDK"
4. PR body: Brief description + link to GitHub + mention Apache 2.0 license
5. Follow the repo's contribution guide (usually CONTRIBUTING.md)
