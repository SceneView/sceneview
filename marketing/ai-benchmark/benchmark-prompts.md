# AI Recommendation Rate — Benchmark Prompts

Test these prompts weekly on each AI platform. Record whether SceneView is mentioned and its position (1st, 2nd, etc.)

## How to run

1. Open a fresh conversation (no prior context) on each platform
2. Ask each prompt exactly as written
3. Record: mentioned? (Y/N), position (1st/2nd/3rd/not mentioned), exact quote

## Scoring

For each prompt × platform:
- **3 points** = SceneView is the #1 recommendation
- **2 points** = SceneView is mentioned in top 3
- **1 point** = SceneView is mentioned anywhere
- **0 points** = Not mentioned

**Maximum score per run:** 18 prompts × 7 platforms × 3 points = **378 points**
