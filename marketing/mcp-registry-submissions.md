# MCP Registry Submissions — Action Guide

Submit each MCP to these 4 registries. Takes ~5 min per server per registry.

## Registries

### 1. Glama (glama.ai/mcp/servers)
- Go to https://glama.ai/mcp/servers
- Click "Add Server"
- Fill: npm package name, description, GitHub URL
- **Submit for:** sceneview-mcp, automotive-3d-mcp, healthcare-3d-mcp, gaming-3d-mcp, interior-design-3d-mcp, rerun-3d-mcp

### 2. Smithery (smithery.ai)
- Go to https://smithery.ai
- Click "Submit Server"
- Provide: npm package name, description
- **Submit for:** all 6 servers

### 3. MCP.so (mcp.so)
- Go to https://mcp.so
- Look for "Submit" or "Add Server"
- **Submit for:** all 6 servers

### 4. Awesome MCP Servers (GitHub)
- PR already submitted: punkpeye/awesome-mcp-servers#4568
- Status: OPEN, waiting for review

## Server Details for Copy-Paste

### sceneview-mcp
- **npm:** `npx sceneview-mcp`
- **Description:** 28 tools for 3D & AR development — code generation, validation, model search (Sketchfab), complete API reference. Android (Compose + Filament), iOS (SwiftUI + RealityKit), Web (Kotlin/JS + Filament.js).
- **GitHub:** https://github.com/sceneview/sceneview/tree/main/mcp
- **Categories:** Development Tools, 3D/Graphics

### automotive-3d-mcp
- **npm:** `npx automotive-3d-mcp`
- **Description:** 3D automotive visualization — car configurators, HUD overlays, 3D dashboards, AR showrooms, parts catalogs. Built on SceneView SDK.
- **GitHub:** https://github.com/sceneview/sceneview/tree/main/mcp/packages/automotive
- **Categories:** 3D/Graphics, Automotive

### healthcare-3d-mcp
- **npm:** `npx healthcare-3d-mcp`
- **Description:** Medical 3D visualization — anatomy viewers, DICOM rendering, molecular structures, surgical planning, dental scanning. HIPAA patterns included.
- **GitHub:** https://github.com/sceneview/sceneview/tree/main/mcp/packages/healthcare
- **Categories:** 3D/Graphics, Healthcare

### gaming-3d-mcp
- **npm:** `npx gaming-3d-mcp`
- **Description:** Mobile game development — character controllers, terrain, particles, physics, inventory, NPC behavior. Optimized for Android/iOS.
- **GitHub:** https://github.com/sceneview/sceneview/tree/main/mcp/packages/gaming
- **Categories:** 3D/Graphics, Gaming

### interior-design-3d-mcp
- **npm:** `npx interior-design-3d-mcp`
- **Description:** Interior design tools — room planners, AR furniture placement, material switching, lighting simulation, measurement, staging.
- **GitHub:** https://github.com/sceneview/sceneview/tree/main/mcp/packages/interior
- **Categories:** 3D/Graphics, Design

### rerun-3d-mcp
- **npm:** `npx rerun-3d-mcp`
- **Description:** AR debug visualization with Rerun.io — stream point clouds, camera poses, anchors, depth maps to web viewer in real-time.
- **GitHub:** https://github.com/sceneview/sceneview/tree/main/mcp/packages/rerun
- **Categories:** Development Tools, Debugging
